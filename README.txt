Instructions:
1. Launch the application from the IDE
2. Enter a username and password on the login screen
3. Select an appointment to modify or delete or
4. Click on Add to create a new appointment
5. Enter the information to create a new Appointment or edit an existing one or
6. Select a customer to modify or delete or
7. Click on Add to create a new customer
8. Enter the information to create a new customer or edit an existing one or
9. Click on report one through report three to see some reports
10. When done, click exit
